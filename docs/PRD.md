ความต้องการของระบบแจ้งซ่อมอุปกรณ์
1. ระบบสมาชิก (User Management)
ผู้ใช้งานสามารถสมัครสมาชิก เข้าสู่ระบบ และออกจากระบบได้ มีการแบ่งระดับผู้ใช้งาน 3 ระดับ ได้แก่ ผู้แจ้งซ่อม ช่างซ่อม และผู้ดูแลระบบ แต่ละระดับเห็นเมนูและสิทธิ์การใช้งานที่แตกต่างกัน
2. ระบบแจ้งซ่อม (Repair Request)
ผู้แจ้งซ่อมกรอกแบบฟอร์มแจ้งซ่อมประกอบด้วยชื่ออุปกรณ์  อาการเสีย ระดับความเร่งด่วน และแนบรูปภาพได้ เมื่อส่งคำร้องแล้วระบบจะสร้างเลขที่คำร้องอัตโนมัติเพื่อใช้ติดตาม
3. ระบบติดตามสถานะ (Status Tracking)
ผู้แจ้งซ่อมสามารถเข้ามาดูสถานะคำร้องของตนเองได้ตลอดเวลา สถานะมี 4 ขั้นตอน ดังนี้
รอดำเนินการ — คำร้องถูกส่งเข้าระบบแล้ว
กำลังซ่อม — ช่างรับงานและเริ่มดำเนินการแล้ว
รอตรวจรับ — ช่างซ่อมเสร็จแล้ว รอผู้แจ้งยืนยัน
เสร็จสิ้น — ผู้แจ้งซ่อมยืนยันรับงานเรียบร้อย
4. ระบบจัดการงานสำหรับช่าง (Technician Work Management)
ช่างเข้าดูรายการงานที่ได้รับมอบหมาย อัปเดตสถานะงาน บันทึกวิธีการซ่อมและอะไหล่ที่ใช้ เมื่อซ่อมเสร็จกดปิดงานพร้อมหมายเหตุ
5. ระบบแจ้งเตือน (Notification)
ระบบส่ง Email อัตโนมัติให้ผู้แจ้งซ่อมทุกครั้งที่สถานะเปลี่ยน และแจ้งเตือนช่างเมื่อได้รับมอบหมายงานใหม่
6. ระบบรายงาน (Report)
ผู้ดูแลระบบดูสรุปภาพรวมได้ เช่น จำนวนงานทั้งหมด งานค้าง งานเสร็จแล้ว และส่งออกเป็น Excel ได้ ต้องการ วิธีใช้งาน Agentic Coding ให้ได้ผลดีที่สุดใน project นี้ต้องสั่งว่าอย่างไร
